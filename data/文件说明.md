## floorplans_unified/ — 统一坐标参数

| 文件 | 大小 | 说明 |
|------|------|------|
| `unified_params.json` | 1.2KB | 平面图统一校正坐标系参数 |

### unified_params.json 结构说明

```json
{
  "origin": "圆形报告厅/圆楼中心",
  "origin_gps": { "lat": 39.731302, "lon": 116.167223 },
  "scale": { "px_per_m": 5, "m_per_px": 0.2 },
  "hall_pixel": { "x": 164, "y": 610 },
  "axes": {
    "x": "向右 = 向东（正值）",
    "y": "向下 = 向南（正值）"
  },
  "formulas": {
    "x_east_m": "(pixel_x - 164) * 0.2",
    "y_south_m": "(pixel_y - 610) * 0.2"
  }
}
```

**用途**：定义统一的像素-米制坐标转换规则，是所有房间坐标校准的基准参数。

## 核心数据文件详解

### wencui_building.json 结构

```json
{
  "building": {
    "name": "文萃楼",
    "description": "北京理工大学良乡校区文萃楼 — 13个分区(A-M)构成的U形教学科研楼宇群",
    "floors": ["1F", "2F", ..., "10F"]
  },
  "nodes": {
    "L-101": {
      "id": "L-101",
      "type": "room",
      "floor": "1F",
      "zone": "L",
      "name": "L-101",
      "x": 409,           // 原始像素 X
      "y": 1695,          // 原始像素 Y
      "x_m": 2.8,         // 米制 X
      "y_m": 12.0,        // 米制 Y
      "ux": 54,           // 统一像素 X
      "uy": 145,          // 统一像素 Y
      "east_m": -22.0,    // 东向米制坐标
      "south_m": -83.8,   // 南向米制坐标
      "ocr_conf": 0.476,  // OCR 置信度
      "ocr_raw": "1-101"  // OCR 原始文本
    }
  },
  "edges": [...],
  "cross_floor_edges": [...]
}
```

**统计**：约 1432 个节点，1165 个房间节点，34636 行 JSON

## 数据使用建议

1. **导航系统开发**：以 `wencui_building.json` 为核心数据源
2. **坐标转换**：参考 `unified_params.json` 中的转换公式
3. **语义解析**：使用 `semantic_mapping.json` 进行自然语言到节点的映射
4. **分区调试**：查看 `zones/` 目录下的分区数据
5. **坐标校准**：参考 `ocr_reports/` 中的 OCR 识别结果
