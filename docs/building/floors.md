# 楼层数据总览

## 空间布局

每层采用统一的 A/B 双区布局：

```
A区(x=30-50)          主走廊(x=100)         B区(x=150-170)
  ┌─楼梯A──┐           ┌────┐           ┌──楼梯B─┐
  │电梯A    │───走廊A───│主走廊│───走廊B───│    电梯B│
  │房间A1-2 │           └────┘           │房间B1-2│
  └────────┘                             └────────┘
```

## 跨层连接

共 36 条跨层边，由 4 条垂直交通链组成：

| 交通链 | 连接楼层 | 边数 |
|--------|---------|------|
| 楼梯 A (stairs_A) | 1F-2F, 2F-3F, ..., 9F-10F | 9 |
| 楼梯 B (stairs_B) | 1F-2F, 2F-3F, ..., 9F-10F | 9 |
| 电梯 A (elevator_A) | 1F-2F, 2F-3F, ..., 9F-10F | 9 |
| 电梯 B (elevator_B) | 1F-2F, 2F-3F, ..., 9F-10F | 9 |
