# 建筑数据

文萃楼建筑数据包含 10 层楼的完整空间拓扑信息，以 JSON 格式存储在 `data/wencui_building.json` 中。

## 数据概览

| 指标 | 数值 |
|------|------|
| 楼层数 | 10 层 (1F-10F) |
| 总节点数 | ~110 个 |
| 楼层内边数 | ~150 条 |
| 跨层边数 | 36 条 |
| 坐标空间 | 200 x 200 |
| 语义映射 | 39 条 |

| 交通链 | 类型 | 位置 |
|--------|------|------|
| 楼梯 A | stairs | A 区 (x=30) |
| 楼梯 B | stairs | B 区 (x=170) |
