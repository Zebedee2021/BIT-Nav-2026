# 建筑数据

文萃楼是北京理工大学良乡校区的大型教学科研综合楼，采用 **U 形布局**，由 13 个分区（A–M）和一座圆形报告厅组成。建筑空间数据以 JSON 格式存储在 `data/wencui_building.json` 中，用于室内导航系统的路径规划。

!!! tip "数据来源"
    本页建筑数据基于文萃楼官方 3D 导航平台的楼层平面图与 GeoJSON 空间数据分析。
    平台地址：`https://wencuigis.info.bit.edu.cn/3dbuilding/#/building2`

## 跨层交通

文萃楼的垂直交通系统按分区分布，而非集中设置：

| 交通设施 | 分布位置 | 覆盖楼层 |
|--------|--------|---------|
| 角楼楼梯 | L、J、C、E 各 2 部 | 1F–10F |
| 角楼电梯 | L、J、C、E 各 1 处 | 1F–10F |
| 主楼楼梯 | A 区北侧 2 部 | 3F–7F |
| 主楼电梯 | A 区电梯厅 | 3F–7F |
| 连接区楼梯 | I、M、F 等区域 | 1F–7F |

每层平均设有 **6+ 处楼梯** 和 **4+ 处电梯厅**，确保各区域均可便捷到达垂直交通。

!!! warning "数据完整性说明"
    当前文档数据基于官方平台的楼层平面图视觉分析。各楼层的详细房间编号已从平面图中提取并记录在[楼层数据总览](floors.md)中。
