# 物理空间建模

## 概述

物理空间建模是三层空间模型的基础层，提供精确的几何与拓扑信息。

## 建模要素

### 几何信息

- **坐标系**：采用 200x200 逻辑坐标空间
- **节点位置**：每个房间、走廊、楼梯、电梯均有 (x, y) 坐标
- **楼层标识**：1F-10F 的楼层编号

### 拓扑信息

- **楼层内连接**：走廊连接房间、楼梯、电梯等设施
- **楼层间连接**：楼梯和电梯构成垂直交通
- **A/B 双区结构**：每层分为 A 区（左侧）和 B 区（右侧）

## 节点分布

```
A区(x=30-50)          主走廊(x=100)         B区(x=150-170)
  ┌─楼梯A──┐           ┌────┐           ┌──楼梯B─┐
  │电梯A    │───走廊A───│主走廊│───走廊B───│    电梯B│
  │房间A1-2 │           └────┘           │房间B1-2│
  └────────┘                             └────────┘
```

## 数据格式

每个节点包含以下属性：

| 属性 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识符 |
| type | string | room/corridor/stairs/elevator |
| floor | string | 所在楼层 |
| x | float | X 坐标 |
| y | float | Y 坐标 |
| name | string | 显示名称 |

## 边的定义

边表示两个节点间的可达关系：

- **同层边**：同一楼层内的连接（权重较低）
- **跨层边**：楼梯/电梯连接不同楼层（权重较高）
