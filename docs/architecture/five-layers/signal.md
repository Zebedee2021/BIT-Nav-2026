# 信号层 (Signal)

## 功能职责

处理和融合传感器数据，输出稳定位姿与轨迹，为空间层提供可靠输入。

## 核心模块

### 位置估计

将原始传感器数据转换为位置信息，支持多种定位方式的初步估计。

### Kalman 滤波

应用 Kalman 滤波器和数字信号处理技术，提高定位精度和稳定性。

### 传感融合

将多个传感器的数据进行时间同步和空间对齐，输出统一的位置估计结果。

## 实施建议

!!! tip "教学原型阶段"
    在教学原型阶段，采用"已知起点"的假设，避免实时定位带来的复杂度。用户通过手动选择或扫码确定当前位置。

## 数据流

```mermaid
graph LR
    Raw["原始观测"] --> PE["位置估计"]
    PE --> KF["Kalman 滤波"]
    KF --> SF["传感融合"]
    SF --> Pos["稳定位姿"]
    Pos --> Space["空间层"]
```
