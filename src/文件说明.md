## 核心导航模块

### pathfinding.py (13KB)

**路径规划模块 - 导航系统核心**

| 类/函数 | 说明 |
|---------|------|
| `Node` | 节点数据类，包含坐标、楼层、区域等信息 |
| `BuildingGraph` | 楼宇图结构，管理节点和边的邻接关系 |
| `AStarPathfinder` | A*路径规划器，实现最短路径搜索 |

**功能**：
- 使用 A* 算法在楼宇拓扑图上搜索最短路径
- 支持同层导航和跨层导航
- 使用欧几里得距离 + 楼层惩罚作为启发式函数

**数据结构**：
```python
@dataclass
class Node:
    id: str
    type: str        # room/corridor/stairs/elevator
    floor: str       # 1F-10F
    zone: str        # A-M
    x: float         # 原始像素坐标
    y: float
    east_m: float    # 米制坐标（东向）
    south_m: float   # 米制坐标（南向）
```

### semantic_mapper.py (8.9KB)

**语义映射模块 - 自然语言理解**

| 类/函数 | 说明 |
|---------|------|
| `SemanticMapper` | 语义映射器，将自然语言映射到节点ID |

**功能**：
- 直接房间号匹配（如 "D402"）
- 语义关键词匹配（如 "教务处" → "D402"）

- 位置关键词识别（如 "东门"、"大厅"）

**依赖数据**：`data/semantic_mapping.json`

### orientation_navigator.py (13.4KB)

**方位感知导航模块 - 实时方向引导**

| 类/函数 | 说明 |
|---------|------|
| `Direction` | 方位枚举（东南西北 + 前后左右） |
| `NavigationDirection` | 导航方向信息数据类 |
| `OrientationNavigator` | 方位导航器 |

**功能**：
- 基于手机指南针的方位导航
- 计算行进方向（东南西北 + 前后左右）
- 生成转向指令（"左转"、"右转"、"直行"）
- 支持多种楼层图朝向配置

**导航输出**：
```python
@dataclass
class NavigationDirection:
    angle: float              # 绝对角度（0-360）
    cardinal_direction: str   # "东"/"南"/"西"/"北"
    relative_direction: str   # "前"/"后"/"左"/"右"
    turn_instruction: str     # "左转进入..."
    distance: float           # 距离（米）
```

## 开发建议

1. **路径规划开发**：重点调试 `pathfinding.py`，确保图结构正确
2. **前端展示**：使用 `floorplan_visualizer.py` + `compass_widget.py`
3. **自然语言输入**：通过 `semantic_mapper.py` 解析用户意图
4. **坐标校准**：使用 `align_floorplans.py` + `recalibrate_rooms.py`
5. **OCR 识别**：使用 `room_label_detector.py` 自动提取房间坐标
