## 一、数据生成工具

### generate_building_graph.py (27.4KB)

**文萃楼建筑导航图数据生成器**

```bash
python tools/generate_building_graph.py
# 输出: data/wencui_building.json
```

**功能**：
- 基于官方楼层图分析生成建筑数据
- 定义分区包围盒（BBOX_L/J/C/E/M/B/I/F/K/H/G）
- 生成节点和边数据
- 创建跨层连接

---

### generate_wencui_room_data.py (32.5KB)

**文萃楼房间节点数据生成器**

```bash
python tools/generate_wencui_room_data.py
```

**功能**：
- 生成 C/D/E/F/G/H/I/J/K/L/M 区的房间结构化数据
- 房间编号规则：`{区号}-{楼层}{序号}`（如 C-101）
- 基于楼层图像素坐标测量

**类**：`WencuiRoomDataGenerator`

---

### generate_complete_room_data.py (31.9KB)

**完整房间数据生成器**

生成包含所有区域的完整房间数据。

---

### merge_room_data.py (8.8KB)

**房间数据合并工具**

合并多个分区数据文件。

## 三、OCR 识别工具

### ocr_rooms.py (11.2KB) ⭐

**OCR 房间位置自动标定**

```bash
python tools/ocr_rooms.py
```

**流程**：
1. 对每层 3000×2000 图像运行 EasyOCR
2. 正则过滤房间号
3. 纠正常见 OCR 误识（L→1、前缀丢失等）
4. 与现有节点列表精确匹配
5. 更新节点坐标

**输出**：
- `data/wencui_building_v6.json`
- `data/ocr_reports/` — 每层识别报告

---

### fix_zones_ocr.py (14.8KB)

**区域 OCR 修正工具**

对 I, J, B, G, A 等低覆盖率区域运行高精度 OCR 修正。

---

## 四、楼层图处理工具

### floorplan_align.py (12.4KB) ⭐

**楼层图统一校正脚本**

**转换流程**：
1. 统一裁切 `(76, 100, 2841, 1820)`
2. 旋转 PIL ROTATE_270（顺时针 90°）
3. 缩放统一 5 px/m，输出 745×1205 px

**坐标系**：
- 原点：圆形报告厅/圆楼中心
- x 轴：向右 = 向东
- y 轴：向下 = 向南

---

### floorplan_to_cad.py (12.8KB)

**楼层图转 CAD**

将楼层平面图转换为 CAD 格式。

**交互式楼层图标注工具**


**操作说明**：
- 左键点击：标注房间中心点

## 工具使用流程

```
1. 生成基础数据
   generate_building_graph.py → wencui_building.json

2. 校准坐标系
   coordinate_system_setup.py → coordinate_system_config.json

3. OCR 识别房间位置
   ocr_rooms.py → wencui_building_v6.json + ocr_reports/

4. 手动校准验证
   room_position_calibrator.py / room_verifier.py

5. 数据迁移清理
   migrate_v6_to_v7.py → wencui_building_v7.json

6. 生成导航案例
   gen_nav_case.py C-101 H-233
```
